# Analyse Prédictive des Ventes Retail

## Description
Ce projet vise à analyser un jeu de données de 10 000+ commandes pour identifier des leviers de rentabilité et prédire la probabilité de réachat client.

## Objectifs
- Nettoyage et exploration (EDA) d'un dataset complexe.
- Entraînement d'un modèle de classification (Scikit-Learn).
- Évaluation de la performance (Précision/Rappel).

## Stack Technique
- Langage : Python
- Bibliothèques : Pandas, Scikit-Learn, Matplotlib, Seaborn
- Outils : Power BI (pour le dashboarding)

## Étapes de réalisation
- [ ] Nettoyage des données (gestion des valeurs manquantes et doublons).
- [ ] Analyse exploratoire et visualisation des tendances.
- [ ] Entraînement du modèle de classification.
- [ ] Déploiement des visualisations KPI.
